Show HN: ContextKit – Agent context management toolkit
https://github.com/seastarbot/contextkit

I've been building AI agents for a while and kept hitting the same wall: context windows filling up, quality degrading after 50% usage, and agents forgetting everything between sessions.

ContextKit is my solution – a drop-in Python library that manages context lifecycle for any agent framework:

- Smart compression: auto-summarize old messages, preserving key info
- Vector indexing: semantic retrieval so you only load relevant context
- Cross-session memory: export/import context between sessions
- Token budget: real-time tracking, auto-cleanup when near limit

The key insight: don't dumb-truncate. Use semantic search to load only what's relevant to the current query.

Works with Claude, GPT, Gemini, any LLM. pip install contextkit.
